# 安全与隐私说明

这个项目涉及求职 JD、简历和模型 API Key，所以隐私边界非常重要。

## 不要提交的信息

请不要把以下内容提交到 GitHub：

- `.env` 文件
- API Key
- 真实姓名、手机号、邮箱、微信号
- 学号、身份证号
- 未公开公司项目或商业数据
- 没有脱敏的真实简历
- 真实面试官姓名或联系方式

## API Key

项目通过 `.env` 读取模型服务 Key：

```env
AI_API_KEY=your_api_key_here
```

`.env` 已经被 `.gitignore` 忽略。请只提交 `.env.example`。

如果你不小心把 API Key 上传到了 GitHub：

1. 立刻删除仓库里的 Key；
2. 去模型服务平台重置或删除旧 Key；
3. 重新生成新的 Key；
4. 检查提交历史中是否仍然能看到旧 Key。

## 真实简历

AI 增强功能会把当前输入发送到你配置的模型服务。使用真实简历前，请先脱敏。

建议替换：

- 姓名 -> 张同学
- 学校 -> 某大学
- 公司 -> 某互联网公司
- 手机/邮箱/微信 -> 删除
- 具体商业数据 -> 改成区间或删除

## 反馈安全问题

如果你发现隐私泄漏、Key 暴露或其它安全问题，请优先通过私下方式联系维护者，不要直接公开贴出敏感信息。
